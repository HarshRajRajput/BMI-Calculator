# BMI-Calculator
A simple and user-friendly BMI (Body Mass Index) Calculator built using HTML ,CSS and  JavaScript. This project helps users calculate their BMI by inputting their weight and height, and provides a classification based on standard BMI categories.
 It take some input from the user and do give the result as follow:
 (1) Allows users to enter their weight and height.
 (2)Computes the BMI based on the provided inputs.
 (3)Categorizes the BMI into standard ranges (e.g., Underweight, Normal weight, Overweight) and display the BMI and the BMI category of the user on the display.
